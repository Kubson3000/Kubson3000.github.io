Micorsoft Windows

Micorsoft Windows, czyli system operacyjny z interfejsem graficznym stworzony przez firme Micorsoft. Częściej zwracany się jako Windows. Posiada on najiększą część marketu systemów operacyjnych dla stacji roboczych.
Systemy firmy Micorsoft są używane na około 75% komputerów. Jako najpopularniejszyą wersję Windows'a oraz najczęściej używany system operacyjny jest Windows 10, który stanowi około 73% marketu wszystkich wersji Windows'a.
